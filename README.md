# MarkMaster-Company
The Purpose of this project is to manage students grades, class , name and contact info depending  on your login credentials. 


Features- View Students feature, Manage Students feature, Add Student feature, Update/ Remove Students feature, Sort Students feature, Login feature, Logout feature, Error feature

View Students- Renders list of students by first and last names

Mane/Sort/Update/Remove/Students Features- Purpose of the manage students feature is meant to edit students details. Purpose pf sprt feature is to sort the students in different ways ( grade in descending or ascending order/ and name in decending or ascending order), Purpose of update/ delete feature is to update students informtaion and remove students.

Update students page- renders page with students informtaion, renders the students id, name, contact (email), grade and the class. You can change/ edit this if log in as admin or teacher. 

Add Student Feature- Purpose is to add students with the appropriate information. The page asks for and id, a name, the contact (email), the grade and the class that you want to add them to.

Login/ Logout Feature- Login feature is intended to login as a registered user, it will ask for a username and a password. Logout feature is intended to logout as the user you are currently logged in as.

Error Feature- Purpose of the error feature is meant to display the error when necessary. For example if you are trying to access something you dont have permissions for such as a student trying to access other students informtiaon or if you try to login and your username or password are incorrect.
